## ⑤工夫した点・こだわった点

- 前回の課題に連動させられるようデザインは統一を意識。
- テキストエリアのメモだけでなく、ボタンクリック後の色の表示も記録できるようにした。

- localStorageに日付情報も連動させた保存と情報の呼び出しを実装させたい。
